# Getting Started

Welcome to **docs-only-example**. This is a documentation-only repository.

## What This Project Contains

- `docs/` — All project documentation
- `docs/adr/` — Architecture Decision Records
- `CONTEXT.md` — Living project context
- `ARCHITECTURE.md` — System architecture overview

## How to Contribute

1. Create a feature branch: `git checkout -b docs/my-topic`
2. Write or update documentation in `docs/`
3. For architectural decisions, create a new ADR in `docs/adr/`
4. Submit a pull request for review

## Writing Guidelines

- Use clear, concise language
- Include code examples where appropriate
- Keep documents focused on a single topic
- Link between documents rather than duplicating content
- Use Markdown for all documentation
